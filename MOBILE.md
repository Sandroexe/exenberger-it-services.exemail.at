## 🎯 Breakpoints

```
Mobile (< 768px)
  - 1 Spalte Layout
  - Große Buttons (volle Breite)
  - Vertikale Navigation
  - Optimierte Schriftgrößen

Tablet (768px - 1024px)
  - 2-3 Spalten Service-Karten

Desktop (> 1024px)

  - 3 Spalten Service-Karten
```

## 🔧 Technische Details

### Viewport & Meta-Tags
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="theme-color" content="#212529">
```

### Touch-Optimierung
- `-webkit-tap-highlight-color: transparent` - Keine Flash auf Tap
- `min-height: 44px` auf Buttons - Apple-Standard
- `:active` statt `:hover` - Für Touchscreen

### Input-Optimierung
- Richtige `type` Attribute (email, tel, text)
- `placeholder` für Hinweise
- `-webkit-appearance: none` - Native Styles entfernen
