# Exenberger IT Services – Statische Webseite

## Hochladen auf GitHub Pages (3 Schritte)

1. Erstelle ein neues Repository auf GitHub (z. B. `meine-firma`).
2. Lade **alle Dateien aus diesem ZIP** in den Repo-Root hoch (Add file → Upload files → drag & drop).
3. Gehe zu **Settings → Pages**, wähle Branch `main` / Folder `/ (root)` und speichern.

Nach 1–2 Minuten ist die Seite live unter `https://<dein-user>.github.io/<repo-name>/`.

## Design anpassen

Alle Farben, Schriften und Abstände stehen ganz oben in `styles.css` unter `:root`.
Eine Änderung dort wirkt auf der gesamten Seite.

## Inhalte anpassen

- `index.html` – Startseite
- `kontakt.html` – Kontaktseite
- `impressum.html` – Impressum (Platzhalter ersetzen!)
- `datenschutz.html` – Datenschutzerklärung
